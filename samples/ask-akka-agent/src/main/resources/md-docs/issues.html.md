# /akka.issues

Convert tasks into GitHub issues.

## <a href="about:blank#_usage"></a> Usage

```none
/akka:issues
```

## <a href="about:blank#_description"></a> Description

Converts existing tasks from `tasks.md` into actionable, dependency-ordered GitHub issues for the feature based on available design artifacts.
