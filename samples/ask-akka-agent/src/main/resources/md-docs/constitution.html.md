# /akka.constitution

Create or update the project constitution.

## <a href="about:blank#_usage"></a> Usage

```none
/akka:constitution
```

## <a href="about:blank#_description"></a> Description

The constitution defines unbreakable project rules regarding technology requirements and design decisions. This command creates or updates `constitution.md` from interactive or provided principle inputs, ensuring all dependent templates stay in sync.

Akka provides a default constitution containing mandatory Akka application requirements. Use this command to layer additional organizational or team-level rules on top.
