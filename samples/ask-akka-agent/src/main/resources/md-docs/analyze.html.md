# /akka.analyze

Cross-artifact consistency and quality analysis.

## <a href="about:blank#_usage"></a> Usage

```none
/akka:analyze
```

## <a href="about:blank#_description"></a> Description

Performs a non-destructive analysis across `spec.md`, `plan.md`, and `tasks.md` to verify consistency and quality. Use this after task generation to confirm that all artifacts are aligned before implementation begins.

This command does not modify any artifacts — it only reports findings.
