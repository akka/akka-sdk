# /akka.checklist

Generate a custom checklist for the current feature.

## <a href="about:blank#_usage"></a> Usage

```none
/akka:checklist
```

## <a href="about:blank#_description"></a> Description

Generates a feature-specific checklist based on user requirements and the current specification. Use this to create a tailored verification list for the feature being developed.
