= akka docker delete-credentials

Delete Docker credentials for given ID.

== Synopsis

The command `akka docker delete-credentials _id_` removes a set of Docker credentials from the project.

TIP: Use the `akka docker list-credentials` command to find the _id_.

----
akka docker delete-credentials ID [flags]
----

== Options

----
  -h, --help             help for delete-credentials
      --owner string     the owner of the project to use, needed if you have two projects with the same name from different owners
      --project string   project to use if not using the default configured project
----
