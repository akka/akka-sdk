= /akka.review

Review implemented code against the spec, plan, and constitution.

== Usage

[source]
----
/akka:review
----

== Description

Reviews the generated code to verify it conforms to the feature specification, implementation plan, and project constitution. Use this after implementation to catch deviations before they compound.
