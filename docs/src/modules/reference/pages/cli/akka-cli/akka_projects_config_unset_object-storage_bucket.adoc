= akka projects config unset object-storage bucket

Remove a single bucket from the object-storage configuration.

== Synopsis

The `akka projects config unset object-storage bucket NAME` command removes a single bucket while leaving the rest of the configuration in place.

Removing the last remaining bucket is not allowed -- the object-storage configuration must contain at least one bucket.
To remove everything, use:

akka projects config unset object-storage

----
akka projects config unset object-storage bucket NAME [flags]
----
