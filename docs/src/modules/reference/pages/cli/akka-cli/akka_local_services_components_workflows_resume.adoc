= akka local services components workflows resume

Resume a suspended workflow execution.

== Synopsis

The `akka local service components workflows resume` command resumes a previously-suspended workflow for the specified service, workflow component and id.

----
akka local services components workflows resume SERVICE_NAME COMPONENT_ID WORKFLOW_ID [flags]
----

== Options

----
  -f, --force            Skip the confirmation prompt.
  -h, --help             help for resume
      --owner string     the owner of the project to use, needed if you have two projects with the same name from different owners
      --project string   project to use if not using the default configured project
      --region string    region to use if project has more than one region
----
