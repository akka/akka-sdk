= akka code context-update

Updates the Akka SDK context directory in the given project directory.

== Synopsis

The `akka code context-update <directory>` command creates or updates the Akka SDK context in the specified project directory by downloading the latest version from the official Akka source, and creates or updates the 'akka-context' subdirectory within the provided project location.

----
akka code context-update [flags]
----
