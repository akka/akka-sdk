= akka services jwks migrate

Migrate JWT keys to JWKS keysets.

== Synopsis

The `akka service jwks migrate` command migrates deprecated JWT keys configured on a service to JWKS keysets.

Keys are grouped by issuer.
Each issuer's keys are combined into a single JWKS secret and a corresponding keyset is added to the service.
Keys with no issuer are grouped together into one keyset with no issuer configured.

The old JWT keys are removed from the service once migration completes.

----
akka services jwks migrate [SERVICE] [flags]
----

== Examples

----
> akka service jwks migrate my-service
> akka service jwks migrate my-service --secret-prefix my-prefix
----

== Options

----
      --force-global           force an existing regional resource to be configured as a global resource
      --force-regional         force an existing global resource to be configured as a regional resource
  -h, --help                   help for migrate
      --owner string           the owner of the project to use, needed if you have two projects with the same name from different owners
      --project string         project to use if not using the default configured project
      --region string          region to use if project has more than one region
      --secret-prefix string   Prefix for generated JWKS secret names (defaults to service name).
----
