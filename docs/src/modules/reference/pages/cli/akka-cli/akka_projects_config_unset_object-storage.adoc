= akka projects config unset object-storage

Unset the object-storage configuration.

== Synopsis

The `akka projects config unset object-storage` command removes the entire object-storage configuration.
To remove a single bucket while keeping the rest of the configuration, use:

akka projects config unset object-storage bucket NAME

----
akka projects config unset object-storage [flags]
----

== Options

----

  -h, --help             help for object-storage
      --owner string     the owner of the project to use, needed if you have two projects with the same name from different owners
      --project string   project to use if not using the default configured project
      --region string    region to use if project has more than one region
