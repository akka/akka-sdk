= akka projects config set object-storage

Set the object-storage configuration.

== Synopsis

The `akka projects config set object-storage` command configures object storage for a project.
Use subcommands for per-bucket edits, or pass `--from-file FILE` to replace the whole configuration from a YAML document with resource: ObjectStorage and a spec.buckets list.

----
akka projects config set object-storage [flags]
----

== Examples

----

akka projects config set object-storage bucket photos \
  --provider s3 --bucket acme-photos --bucket-region us-east-1 \
  --credentials-type static \
  --access-key-id-secret aws-creds/access-key-id \
  --secret-access-key-secret aws-creds/secret-access-key

akka projects config set object-storage --from-file project-os.yaml
----

== Options

----
      --force-global       force an existing regional resource to be configured as a global resource
      --force-regional     force an existing global resource to be configured as a regional resource
      --from-file string   path to a YAML descriptor containing an ObjectStorage document
  -h, --help               help for object-storage
      --owner string       the owner of the project to use, needed if you have two projects with the same name from different owners
      --project string     project to use if not using the default configured project
      --region string      region to use if project has more than one region
----
