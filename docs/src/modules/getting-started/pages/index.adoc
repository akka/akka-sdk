= Getting Started

Akka is an agentic AI platform for building distributed, resilient, and intelligent applications using autonomous, stateful agents. This section will help you take your first steps, whether you're exploring Akka for the first time, evaluating its capabilities, or preparing to build a production system.

== Choose your starting point

=== 🛠 Hands-on walkthroughs

Get up and running fast with our curated, interactive experiences:

* xref:java:author-your-first-service.adoc[Author your first agentic service]:  
  Build from scratch using the Akka SDK.

* xref:java:shopping-cart/quickstart.adoc[Explore a complete example]:  
  Study a real-world shopping cart system.

* https://console.akka.io/register[Try Akka in the browser]:  
  Deploy, test, and replicate services without installing anything.

== What you'll build

As you proceed, you'll learn how to:

* Define stateful agent entities and workflows
* Persist and replicate state across regions
* Integrate with LLMs and external tools
* Observe reasoning paths and decision traces
* Evaluate system behavior using Akka EvalOps
