= Getting Started

== Tutorials

These hands-on tutorials walk you step by step through building real Akka systems, from your first minimal service to fully featured applications like shopping carts and AI agents. Each tutorial includes guided instructions, runnable code, and explanations of key concepts.

=== Build your first service

xref:java:author-your-first-service.adoc[Author your first agentic service]  
Create and run a minimal Akka-based service from scratch using the Maven archetype. Ideal for newcomers looking to get hands-on experience fast.

=== Build a shopping cart system

Explore a complete e-commerce service and learn key Akka concepts by implementing a real-world system:

* xref:java:shopping-cart/index.adoc[Part 1: Build a simple cart service]
* xref:java:shopping-cart/lookup.adoc[Part 2: Authenticated user-specific lookup]

These lessons walk you through defining agents, handling state, processing commands, and responding to user-specific queries.

=== Build an AI RAG Agent

Learn how to implement a Retrieval-Augmented Generation (RAG) pipeline with Akka:

* xref:java:ask-akka/part1-session.adoc[Part 1: Modeling the agent session]
* xref:java:ask-akka/part2-indexing.adoc[Part 2: Knowledge indexing with a workflow]
* xref:java:ask-akka/part3-queries.adoc[Part 3: Executing RAG queries]
* xref:java:ask-akka/part4-streaming.adoc[Part 4: Adding streaming endpoints]

This series covers end-to-end design of a multi-agent system that performs LLM-assisted reasoning, indexing, and live querying.
