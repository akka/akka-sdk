= Projects

include::ROOT:partial$include.adoc[]

Akka services are deployed to projects owned by an xref:operations:organizations.adoc[organization] in a xref:operations:regions.adoc[region].

== Details

* From a security standpoint, projects are isolated from other projects.
* Users are given access to projects on a per-project basis.
* One or more services can be deployed to a project.
* A project is located in a region.
