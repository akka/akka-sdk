NOTE: In case there is any trouble with installing the CLI when following these instructions, please check the xref:operations:cli/installation.adoc[detailed CLI installation instructions].

[.tabset]
Linux::
+
--
Install the `akka` CLI using the Debian package repository:

[source,bash]
----
curl -1sLf \
  'https://downloads.akka.io/setup.deb.sh' \
  | sudo -E bash
sudo apt install akka
----

--
macOS::
+
--
The recommended approach to install `akka` on macOS, is using https://brew.sh[brew, window="new"]

[source,bash]
----
brew install akka/brew/akka
----

--
Windows::
+
--
Install the `akka` CLI using https://learn.microsoft.com/en-us/windows/package-manager/winget/[winget]:

[source,powershell]
----
winget install Akka.AkkaCli
----
