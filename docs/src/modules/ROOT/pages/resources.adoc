== Foundational Works

* link:{url-ext-reactive-manifesto}[The Reactive Manifesto, window="new"] — By Jonas Boner, Dave Farley, Roland Kuhn, Martin Thompson. Four pillars: Responsive, Resilient, Elastic, Message Driven. The foundational philosophy behind Akka.
* link:{url-ext-reactive-principles}[The Reactive Principles, window="new"] — By Jonas Boner et al. 8 principles and 6 patterns for cloud-native and edge-native design. Companion to the Reactive Manifesto.
* **Reactive Design Patterns** (Manning) — By Roland Kuhn, Brian Hanafee, Jamie Allen. Patterns for building reactive systems.
