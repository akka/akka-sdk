= Integrations
include::ROOT:partial$include.adoc[]

Akka enables seamless communication between services, components and external systems. The following integration options are available:

* xref:java:component-and-service-calls.adoc[Component and Service Calls]: Directly invoke other services or components within your application.
* xref:java:message-brokers.adoc[Message Brokers]: Publish to and subscribe to events from external message brokers like Kafka or Pulsar.
* xref:java:streaming.adoc[Streaming]: Built-in support for processing and reacting to streams of data.
